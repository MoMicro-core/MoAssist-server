MoMicro Assist

Firebase auth for your dashboard and FCM-aware backend sessions.

This page signs the user in with Firebase, reads the Firebase ID token, optionally collects an FCM token, and creates the MoMicro Assist backend session through `/v1/auth/session`.

Register new customers with Firebase and immediately provision a MoMicro Assist user session.
Log in existing customers and persist the backend token for dashboard calls and websocket auth.
If Messaging is configured, the page will also attach an FCM token to the backend session request.
Or continue with email

The created backend session token is stored in `localStorage` under `momicro-assist-session`.
